## 🚀 What I’m Building Right Now

I’m currently focused on building and publishing projects that explore modern data platform patterns, scalable architectures, and real-world engineering tradeoffs.

### Active Projects

- **identity-risk-streaming-lakehouse**  
  A streaming-first security analytics pipeline built in Databricks using a medallion architecture (Bronze, Silver, Gold) on identity-related login event data. Focus areas include event ingestion, data quality, anomaly detection, and dashboard-ready risk analytics.

- **streaming-pipeline-demo**  
  An end-to-end event processing project focused on ingestion, transformation, windowed analytics, and streaming design patterns inspired by modern data platform stacks.

- **data-platform-capstone**  
  A broader platform reference project that brings together ingestion, transformation, governance, reporting, and operational design into a unified data platform architecture.

- **postgres-perf-lab**  
  Reproducible experiments focused on indexing, partitioning, query planning, and database performance tradeoffs.

- **rag-postgres**  
  A practical retrieval pipeline using PostgreSQL and pgvector to explore production-oriented RAG design, structured retrieval, and API-driven serving patterns.
